Quick read:

  • A · icon + colored text + slash. Strong color signal, still feels lightweight.
  • B · only the icon is colored, text stays neutral. Cleanest scan — eye picks up the icons first, reads names if it cares.
  • C · just icons (no text). Most compact, requires the user to learn icon→category map. Title-attribute reveals on hover.
  • D · full pills with icon inside. Maximum signal, maximum noise — risk of fruit-salad at 3+ labels.

My read: B is the strongest balance — the icon does the color signaling so the text can stay in --text-secondary and the row stays calm. Option A is the second-best if you want the color to "louder."
