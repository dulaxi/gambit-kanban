What changed vs the current build

  • Width480px (was 640px). Closer to a Notion / claude.ai dialog than a configurator.
  • Heading"New board" at 24px serif (was 28px + a subtitle). One word, no marketing copy.
  • ComposerNo more pill border — just an underlined field that turns ink-black on focus. Less chrome, more "writing a name."
  • TemplatesReplaced the 3×2 chip grid with a one-line "Columns: A · B · C · D · Change". The picker only appears when you click Change. Default state has zero template chrome.
  • Picker UIWhen opened, a quiet vertical list with template name + actual column names — no mini-board previews. Selected row gets lime-wash + check.
  • ButtonsInk primary (was lime accent). Lime is louder; ink reads more like claude.ai's actions. Cancel demoted to a ghost text link.
  • Spacing56px top padding inside the modal (was 40px). 32px between heading and composer (was 28px). Generally more whitespace.
