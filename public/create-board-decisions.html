DIRECTION · F

Notion-Inline no modal

Skip the modal entirely. "+ New board" expands inline at the top of the boards list. Title input is editorial-large, template chips reveal as a single quiet row. Hitting Enter creates and immediately routes you in. Closest to how Notion / Linear handle it. Lowest-friction option.

fastest path no context switch lower visual ceremony templates feel optional rather than featured

How to think about choosing

If you want polished + safe, A is shipped and works. If you want more impressive on first open, B (Showroom) gives the most "this app cares about my time" feel without new infra. If you want to signal AI as the differentiator, C is the only direction that does that — but it requires wiring an actual suggestion call. D is for editorial vibes / screenshots; E is for breaking the modal pattern entirely; F is for "actually I think modals are wrong here."
