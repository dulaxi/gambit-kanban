Recently fixed

::selection text color fixed

.ai-skeleton-block gradient fixed
Hardcoded `#E8DDE2` and `#EEF2D6` → `var(--color-mauve-wash)` and `var(--color-lime-wash)`. Skeleton now shimmers with theme-appropriate bands.
.ai-shimmer-reveal anchor fixed
First gradient stop changed from `#1B1B18` → `var(--text-primary)`. Revealed text reads correctly in both themes; rainbow ribbon kept intact.
Default scrollbar thumb fixed
New tokens `--scrollbar-thumb` and `--scrollbar-thumb-hover` replace the body-level hardcoded `rgba(27,27,24,...)`. Visible on both themes.
Calendar "today" pill fixed
Pill text changed from hardcoded `text-white` to `text-[var(--btn-primary-text)]` — pairs correctly with `--color-bark` in both themes.
Chat user bubble fixed
Inline `style={{ background: '#E8DDE2' }}` removed — bubble now uses `bg-[var(--color-mauve-wash)]`.
Markdown code block fixed
Dropped the ink-on-cream inversion. Block code now uses `bg-[var(--surface-raised)] text-[var(--text-primary)]` for consistency with inline `code`.
Chat composer shadow fixed
Three hardcoded multi-layer rgba shadows replaced with `--chat-input-shadow`, `--chat-input-shadow-hover`, `--chat-input-shadow-focus`. Punchier shadow alpha and stronger hairline ring on dark surfaces.

`AVATAR_COLORS` no longer carries hardcoded `#D6E8E0`. Promoted to `--color-mint-wash` with a dark equivalent of `#1F3030` (dark teal).
