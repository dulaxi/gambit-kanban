Bottom line for CLAUDE.md

  • One canonical primitive exists today: Avatar. Everything else needs extracting from existing inline usages.
  • Token system is healthy — light + dark variants in src/index.css already used everywhere.
  • Coherency phase work: extract Button → Input → Modal → Menu → Tooltip → Toast → Skeleton from feature components, in roughly that priority order based on usage frequency.
